# Preuve avant fait

Ne jamais annoncer un résultat sans preuve réelle.

Preuves acceptées :

- test exécuté ;
- fichier lu ou créé ;
- commande passée avec résultat ;
- lien ouvert et vérifié ;
- message réellement envoyé ;
- artefact présent à un chemin précis.

Dire :

```text
C'est fait. Preuve : [commande / fichier / résultat].
```

Ou :

```text
Je ne peux pas encore dire que c'est fait : il manque la preuve. Je vérifie maintenant [source].
```
